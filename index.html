Complete Step-by-Step Instructions for Seven-Segment Display Encoder
1
Setup Your Logisim Project
  • Open Logisim software on your computer
  • Go to File ‚Üí Save As
  • Save the file as YourLastName.circ
  • Go to Project ‚Üí Add Circuit
  • Type Encoder as the Circuit Name
  • Click OK
üí° Tip: You should now see two circuits in your project explorer: "main" and "Encoder". We'll work on the Encoder circuit first.
2
Add Input Pins to Encoder Circuit
  • Double-click on Encoder in the project explorer to open it
  • From the toolbar, select the Pin tool (or go to Wiring ‚Üí Pin)
  • Place 4 pins on the LEFT side of the canvas, vertically spaced
  • Click on each pin and set its properties:
    • Top pin: Label = "A", Output = No
    • Second pin: Label = "B", Output = No
    • Third pin: Label = "C", Output = No
    • Bottom pin: Label = "D", Output = No
3
Add NOT Gates (Only 4!)
  • From Gates menu, select NOT Gate
  • Place 4 NOT gates next to each input pin
  • Connect each input (A, B, C, D) to its corresponding NOT gate
  • This gives you: A, A', B, B', C, C', D, D' (8 signals total)
‚ö†Ô∏è IMPORTANT: The project requires EXACTLY 4 NOT gates. Do not add more! You'll reuse these inverted signals throughout your circuit.
4
Implement Output 'a' Expression
a = AB'C' + A'BD + AD' + A'C + BC + B'D'

This expression has 6 product terms (AND operations) connected by OR.

  • Create 6 AND gates (one for each product term):
    • AND gate 1: Connect A, B', C' (3 inputs)
    • AND gate 2: Connect A', B, D (3 inputs)
    • AND gate 3: Connect A, D' (2 inputs)
    • AND gate 4: Connect A', C (2 inputs)
    • AND gate 5: Connect B, C (2 inputs)
    • AND gate 6: Connect B', D' (2 inputs)
  • Create 1 OR gate with 6 inputs
  • Connect all 6 AND gate outputs to the OR gate inputs
  • Connect the OR gate output to an output pin labeled "a"
üí° Tip: To set the number of inputs on an AND or OR gate, click on the gate and look at the properties panel. Change "Number of Inputs" to match what you need (2, 3, 4, 5, or 6).
5
Implement Output 'b' Expression
b = A'C'D' + A'CD + AC'D + B'C' + B'D'
  • Create 5 AND gates:
    • AND gate 1: A', C', D' (3 inputs)
    • AND gate 2: A', C, D (3 inputs)
    • AND gate 3: A, C', D (3 inputs)
    • AND gate 4: B', C' (2 inputs)
    • AND gate 5: B', D' (2 inputs)
  • Create 1 OR gate with 5 inputs
  • Connect all AND outputs to the OR gate
  • Connect OR output to output pin labeled "b"
6
Implement Output 'c' Expression
c = A'C' + A'D + C'D + (A ‚äï B)
  • Create 3 AND gates:
    • AND gate 1: A', C' (2 inputs)
    • AND gate 2: A', D (2 inputs)
    • AND gate 3: C', D (2 inputs)
  • Create 1 XOR gate: Connect A and B
  • Create 1 OR gate with 4 inputs
  • Connect the 3 AND outputs + XOR output to the OR gate
  • Connect OR output to output pin labeled "c"
üí° Note: The ‚äï symbol means XOR. Find the XOR gate in Gates ‚Üí XOR Gate.
7
Implement Output 'd' Expression
d = A'B'D' + B'CD + BC'D + BCD' + AC'
  • Create 5 AND gates:
    • AND gate 1: A', B', D' (3 inputs)
    • AND gate 2: B', C, D (3 inputs)
    • AND gate 3: B, C', D (3 inputs)
    • AND gate 4: B, C, D' (3 inputs)
    • AND gate 5: A, C' (2 inputs)
  • Create 1 OR gate with 5 inputs
  • Connect all AND outputs to the OR gate
  • Connect OR output to output pin labeled "d"
8
Implement Output 'e' Expression
e = B'D' + CD' + AC + AB
  • Create 4 AND gates:
    • AND gate 1: B', D' (2 inputs)
    • AND gate 2: C, D' (2 inputs)
    • AND gate 3: A, C (2 inputs)
    • AND gate 4: A, B (2 inputs)
  • Create 1 OR gate with 4 inputs
  • Connect all AND outputs to the OR gate
  • Connect OR output to output pin labeled "e"
9
Implement Output 'f' Expression
f = A'BC' + C'D' + BD' + AB' + AC
  • Create 5 AND gates:
    • AND gate 1: A', B, C' (3 inputs)
    • AND gate 2: C', D' (2 inputs)
    • AND gate 3: B, D' (2 inputs)
    • AND gate 4: A, B' (2 inputs)
    • AND gate 5: A, C (2 inputs)
  • Create 1 OR gate with 5 inputs
  • Connect all AND outputs to the OR gate
  • Connect OR output to output pin labeled "f"
10
Implement Output 'g' Expression
g = A'BC' + B'C + CD' + AB' + AD
  • Create 5 AND gates:
    • AND gate 1: A', B, C' (3 inputs)
    • AND gate 2: B', C (2 inputs)
    • AND gate 3: C, D' (2 inputs)
    • AND gate 4: A, B' (2 inputs)
    • AND gate 5: A, D (2 inputs)
  • Create 1 OR gate with 5 inputs
  • Connect all AND outputs to the OR gate
  • Connect OR output to output pin labeled "g"
11
Add Output Pins
  • On the RIGHT side of your circuit, add 7 output pins
  • Click each pin and set Output = Yes
  • Label them from top to bottom: a, b, c, d, e, f, g
  • Connect each expression's final OR gate to its corresponding output pin
12
Test Your Encoder
  • In the Encoder circuit, add a 7-Segment Display component (I/O ‚Üí 7-Segment Display)
  • Connect outputs a, b, c, d, e, f, g to the corresponding pins on the display
  • Add input pins or a splitter to test different values
  • Test with values 0-9 and verify correct digit display
üí° Testing: Use the Poke tool (hand icon) to change input values and verify the display shows the correct digit!

13
Customize Circuit Appearance
  • Right-click on Encoder in the project explorer
  • Select Edit Circuit Appearance
  • You'll see a rectangular box with pins on the sides
  • Use the Text tool to add "Encoder" label inside the box
  • Arrange the pins neatly (4 inputs on left, 7 outputs on right)
  • Click back to the main window when done
14
Create Main Circuit
  • Double-click on main (your last name circuit) in the project explorer
  • From the project explorer, drag the Encoder component onto the canvas
  • Add 4 input pins on the left, label them A, B, C, D
  • Connect these to the Encoder's inputs
  • Add a 7-Segment Display from I/O menu
  • Connect Encoder outputs (a-g) to the display's corresponding pins
15
Final Testing & Submission
  • Use the Poke Tool (hand icon) to test all inputs 0-9
  • Verify each digit displays correctly on the 7-segment display
  • Add a text label with your team members' names and ID numbers
  • Save your file: File ‚Üí Save
  • Test one more time to ensure everything works
  • Demonstrate to your instructor during lab hours
  • Submit the .circ file to Moodle
üí° Final Checklist:
  • ‚úì Encoder circuit has exactly 4 NOT gates
  • ‚úì All 7 Boolean expressions implemented correctly
  • ‚úì 4 input pins (A, B, C, D) labeled
  • ‚úì 7 output pins (a, b, c, d, e, f, g) labeled
  • ‚úì Main circuit has Encoder connected to 7-Segment Display
  • ‚úì All digits 0-9 display correctly when tested
  • ‚úì Team members' names and IDs added as text
  • ‚úì File saved with your last name

g = A'BC' + B'C + CD' + AB' + AD

  • Label ALL inputs and outputs clearly
  • Test with ALL values 0-9 before submitting
